## ğŸ¤– How It Works

The application leverages the **Google Gemini API** (`gemini-2.5-flash` and `gemini-2.5-flash-image`) to dynamically hallucinate entire corporate environments.

1.  **Context Injection**: We feed the model specific company profiles (industry, typical problems).
2.  **Structured Generation**: The AI returns strictly typed JSON data for tasks, ensuring the UI remains stable while the content remains dynamic.
3.  **Asset Creation**: When a user opens a task, the AI generates the specific "files" (data tables, images) on the fly, making every simulation unique.
